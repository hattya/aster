vimscript.covimerage(...args)
-----------------------------

``vimscript.covimerage`` represents ``coimerage``, and its commands are
defined as properties of this function.

Requirements
  - `covimerage <https://github.com/Vimjas/covimerage>`_

~~~~~~~~~~~~~~~~~~~~~~~~~~
